Hue Group

Der Baustein steuert Hue Gruppen und liest den Status der Gruppe oder einzelner Lampen aus. Um die Netzwerklast zu verringern können mehrere Bausteine als Kaskade geschaltet werden, um die Statusmeldung der Hue-Bridge für mehrere Lampen bzw. Gruppen auszulesen.

Baustein Id: 14100
Kategorie: Datenaustausch
Gültigkeit der Hilfe: Bausteinversion 0.5

Eingänge

Eingang Initwert Beschreibung
Groups Json Für Baustein-Kaskade: Verknüpfung des gleichnamigen Ausgangs eines anderen Bausteins.
Lights Json Für Baustein-Kaskade: Verknüpfung des gleichnamigen Ausgangs eines anderen Bausteins.
Get Status Ein Wert =1 triggert den Status-Abruf über die Hue-Bridge. Hier ist es z.B. Sinnvoll einen Telegrammgenerator anzuschließen. Damit kann detektiert werden, wenn eine Lampe auf einem anderen Weg, als über den Baustein geschaltet wird.
Hue Bridge IP IP der Hue Bridge
Hue Bridge Port 80 Port der Hue Bridge
User Hue Bridge User
Group Id 0 Id der zu schaltenden Hue Gruppe, vgl. http://hue-ip/api/hue-user/groups
Light Id 0 Id der auszuwertenden Hue Lampe, vgl. http://hue-ip/api/hue-user/lights
Grp On/Off 0
Grp Brightness 0 Helligkeit für die Hue Gruppe in Prozent.
Grp Hue Color 0 Hue Farbe der Hue Color.
Grp Saturation 0 Sättigung der Hue Gruppe.
Grp Hue Scene Hue Scene für die Hue Gruppe, vgl. http://hue-bridge-ip/user/scenes

Ausgänge

Ausgang Initwert Beschreibung
Grp On/Off 0 Ein/Aus Status
Grp Brightness 0 Aktuelle Helligkeit in Prozent
Grp Hue Color 0 Aktuelle Hue Color
Grp Saturation 0 Aktuelle Sättigung
Lght Reachable 0 Erreichbarkeit der Lampe (nicht Gruppe!)
Groups Json Json Struktur für den Status der Gruppe für Kaskaden von Bausteinen (für den Eingang des Folgebausteins).
Lights Json Json Struktur für den Status der Leuchten für Kaskaden von Bausteinen (für den Eingang des Folgebausteins).
