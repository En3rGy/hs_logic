DWD Unwetter

Der Baustein liest die JSON-Daten zu aktuellen Unwetterwarnungen des DWD aus und stellt die Informationen zur Verfügung. Bei mehreren Warnungen wird die höchste Unwetterwarnung ausgegeben - unabhängig vom Warnzeitraum.

Baustein Id: 14101
Kategorie: Datenaustausch
Gültigkeit der Hilfe: Bausteinversion 0.11

Eingänge

Eingang Initwert Beschreibung
Trigger 0 Bei einem Wert =1 wird die aktuelle DWD-Json datei abgerfen und ausgewertet.
Region Id Id der Region / Stadt / Landkreis aus der DWD-Json Datei, für die die Unwetterdaten ausgegeben werden sollen.

Ermittlung der ID

Die DWD-Json Datei listet aktuell die Unwetterwarnungen auf regionaler Ebene. Um die Id der eigenen Region zu erhalten, muss man:

  1. Öffnen der DWD JSON-Daten im Browser (Link anklicken)
  2. Suchen der gewünschten Region / Stadt / Landkreis
  3. In der Datei sieht die Struktur z.B. wie folgt aus:
    "103401000":[{"regionName":"Stadt Delmenhorst","start":1580...
    Die Id von "Stadt Delmenhorst" ist dabei "103401000".
Region Name der Region / Stadt / Landkreis aus der DWD-Json Datei, für die die Unwetterdaten ausgegeben werden sollen. Innerhalb der Json-Datei wird nach dem 1. Vorkommen des Namens gesucht. "Frankfurt" liefert die Ergebnisse für Frankfurt am Main oder Frankfurt an der Oder

Ausgänge

Ausgang Initwert Beschreibung
Überschrift Überschrift der Unwetetrwarnung.
Level 0 Level der höchten Unwetterwarnung für die Gemeinde / Stadt.
Beschreibung Beschreibung der Unwetterwarnung,
Hinweise Handlungsempfehlungen für die Unwetterwarnung.
Startzeit 0 Start-Zeit des Warnfensters in s als UNIX-Zeit.
Stoppzeit 0 Stop-Zeit des Warnfensters in s als UNIX-Zeit.
Text alle Warnungen Text, der alle aktuell gemeldeten Warnereignisse auflistet. Leer, wenn keine Warnungen vorliegen.
Text Wetterwarnung (Lv 1) Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis dem DWD Level 1 (=2 am Ausgang Level) entspricht. Sonst wird ein leerer Text ausgegeben.
Text markantes Wetter (Lv 2) Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis dem DWD Level 2 (=3 am Ausgang Level) entspricht. Sonst wird ein leerer Text ausgegeben.
Text Unwetterwarnung (Lv 3) Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis dem DWD Level 3 (=4 am Ausgang Level) entspricht. Sonst wird ein leerer Text ausgegeben.
Text extremem Unwetter (Lv 4) Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis dem DWD Level 4 (=5 am Ausgang Level) entspricht. Sonst wird ein leerer Text ausgegeben.
Text Vorabinformation Unwetter Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis einer Vorwarnung entspricht (=1 am Ausgang Level). Sonst wird ein leerer Text ausgegeben.
Text Hitzewarnung Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis einer Hitzewarnung entspricht (=10 am Ausgang Level). Sonst wird ein leerer Text ausgegeben.
Text UV-Warnung Gibt eine Liste aller Warnereignisse aus, wenn das höchste Warnereignis einer UV-Warnung entspricht (=20 am Ausgang Level). Sonst wird ein leerer Text ausgegeben.
Warnung aktiv (sbc) 0 1 wenn die aktuelle Zeit im Warnfenster liegt, also die Warnung akut ist.
Error 0 1 wenn ein Fehler vorliegt. Die HS-Debug-Seite (hs-ip/hslist?lst=debug)liefert in der Kategorie HSL 2.0 und der Unterkategorie hsl20_3_dwd mehr Details.
